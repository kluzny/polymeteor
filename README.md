## Getting Started

1. Make sure you have [node via nvm](https://github.com/creationix/nvm) and [meteor](https://www.meteor.com/install) installed
2. Clone down our repo ( or your own fork if your feeling froggy )
3. cd polymeteor && meteor
4. That's it!
